TLOS-003 · Description

Fragment-based writing app. Characters, lore, and prose compose into structured LLM context.

GPL-2.0 · 2026.06
SPEC 14 ENTRIES
Fragment system
Tagged, refable units with sticky / system placement and version history.
Prose chain
Branchable prose with regenerate, refine, switch, remove. Selection transforms: rewrite, expand, compress.
Prewriter
Optional planner pass distills full context into a focused brief before generation.
Per-role models
Separate provider and model for generation, librarian, character chat, directions, and per-agent overrides. Automatic fallback.
Model instructions
Drop JSON files to override any system prompt by model family. Exact match or regex.
Character Chat
Story-scoped chat with streaming, portraits, and saved history.
Block context
Visual editor for reordering, overriding, and extending the LLM prompt. JavaScript script blocks with live preview.
Librarian memory
Rolling continuity, hierarchical summaries, contradiction tracking, summary compaction.
Import / export
JSON, ZIP bundles, SillyTavern / Tavern cards with lorebook support.
Plugins
Bundled and external runtime plugins, iframe UI panels. SDK: @tealios/errata-plugin-sdk.
Storage
Filesystem. No database. Compiles to a single binary.
Stack
Bun, TanStack Start, React 19, Elysia, Vercel AI SDK, Tailwind v4, shadcn/ui.
Platforms
Windows, Linux, macOS. Standalone binary, no runtime required.
