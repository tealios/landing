TLOS-001 · Description

Phone camera streams to a desktop viewer over WebRTC. Pair by QR, route into OBS.

MIT · 2026.06
SPEC 11 ENTRIES
Pairing
QR code, browser to desktop, single room per session.
Transport
WebRTC peer-to-peer, video and audio.
Signaling
Express + WebSocket, runs in-process on the desktop app.
Sender
Progressive web app in the mobile browser. No install.
Viewer
Electron desktop app. Single window.
Camera controls
Resolution, framerate, bitrate, audio toggle, front/back swap, all remote.
OBS source
16:9 chromeless capture window.
Off-LAN
Bundled cloudflared tunnel, optional.
Idle behavior
Screen wake lock on phone, REC indicator on viewer.
Platforms
Windows (NSIS), macOS (Intel, Apple silicon).
License
MIT.
